# Campus Companion

A React Native mobile application built with Expo for students at the College of Science and Technology (CST), Bhutan. The app provides quick access to important campus contacts, class schedules, and notice board announcements.

## Features

- Home screen with quick navigation links
- Contacts screen with important campus contacts
- Schedule screen with weekly class timetable
- Notice Board screen with latest announcements
- Tap any contact to view full details

## Tech Stack

- React Native with Expo
- TypeScript
- React Navigation (Stack + Bottom Tabs)

## Installation and Running

1. Clone the repository:
   ```
   git clone <your-repo-url>
   ```

2. Navigate to the project folder:
   ```
   cd Assignment1/CampusCompanion
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Install React Navigation:
   ```
   npm install @react-navigation/native @react-navigation/native-stack @react-navigation/bottom-tabs
   ```

5. Install required Expo packages:
   ```
   npx expo install react-native-screens react-native-safe-area-context
   ```

6. Start the app:
   ```
   npx expo start
   ```

7. Scan the QR code with the Expo Go app on your phone, or press `a` for Android emulator, `i` for iOS simulator.

## Known Issues and Limitations

- All data is hardcoded, no backend or database is used
